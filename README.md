## üöÄ Getting Started

1. Clone the repository:

```bash
git clone https://github.com/adithpv/shadcn-project.git
cd shadcn-project
```

2. Install dependencies:

```bash
npm install
# or
yarn install
# or
pnpm install
# or
bun install
```

3. Run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

4. Open [http://localhost:3000](http://localhost:3000) to explore the components.
